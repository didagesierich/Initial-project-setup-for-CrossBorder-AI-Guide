# CrossBorder AI Guide

AI prototype that analyzes cross-border scenarios and generates a procedural roadmap.

## What it does

The agent interprets a user scenario and generates:

* required documents
* institutions involved
* procedural steps
* a simplified process flow

## How to run the demo

1. Download the repository
2. Open the project folder
3. Double-click **index.html**
4. Enter a cross-border scenario
5. Click **Analyze Procedure**

The system generates a procedural roadmap.

## Example scenario

"I want to export aluminum kitchen products from India to the EU market."

The agent produces a simplified cross-border procedure analysis.
