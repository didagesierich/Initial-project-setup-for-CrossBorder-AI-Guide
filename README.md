# CrossBorder AI Guide

CrossBorder AI Guide is an AI agent that helps users understand cross-border administrative or market-entry scenarios.

The system interprets a scenario, asks clarification questions, maps relevant procedures or regulatory domains, and generates a structured action roadmap.

## How to run the project

1. Open the file **index.html** in a web browser.
2. Enter a cross-border scenario.
3. Click **Analyze Scenario**.
4. The AI agent will interpret the scenario and generate a Cross-Border Procedure Roadmap.

## Example scenario

Input:
Romanian citizen marrying in Germany

Output:
A structured roadmap including:
- required documents
- responsible institutions
- complexity level
- step-by-step actions
